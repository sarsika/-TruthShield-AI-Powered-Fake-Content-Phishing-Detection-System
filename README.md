# 🛡️ TruthShield

### AI-Powered Fake Content & Phishing Detection Platform

🔍 Detect fake news, phishing URLs, and suspicious emails using AI-powered analysis.

---

## 📌 Project Overview

Every day, users are exposed to fake news, phishing attacks, and malicious emails, leading to misinformation, identity theft, and financial risks.

TruthShield addresses these challenges by providing an intelligent platform that analyzes multiple types of digital content using AI-powered prediction models.

The application provides a secure and intuitive interface, enabling users to analyze digital content efficiently through a scalable backend architecture and AI-powered prediction models.

## 🔄 Workflow

1. 👤 User registers or logs into the application  
2. 🔐 JWT authentication validates user access  
3. 📝 User submits news, URL, or email content  
4. ⚙ Backend processes request via REST APIs  
5. 🤖 AI/ML models analyze the content  
6. 📊 Prediction result is generated  
7. 💾 Result stored in MongoDB database
